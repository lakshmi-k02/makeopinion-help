Enable Soft Launch for an Audience

Enable Soft Launch +

How to create from Create Survey?

Click Start a Project in the upper-right corner of your account and choose one of the options: Get Survey Participants, Start a new survey, Select Survey Templates.

How to create from Edit Survey?

Follow these steps to enable the soft launch feature for a specific audience:

    Access the Project: Navigate to the project where you want to enable the soft launch feature.
    Select the Audience: Open the audience you want to configure for the soft launch.
    Edit Audience Settings:
        Go to Edit Audience.
        Navigate to the Scheduling tab.
    Enable Soft Launch:
        Locate the "Enable Soft Launch" option and activate it.
        Choose the soft launch type (e.g., clicks or completes).
        Specify the number of clicks or completes required.
    Save Changes: Click Save to apply your settings.

Once saved, the soft launch configuration is active.
