Soft Launch

Limit project exposure by sending to a small group first before going live fully.

What It Does

Lets you test a project with a limited number of respondents to catch setup issues early.

How to Enable

  • Go to Launch Settings.
  • Toggle Soft Launch to ON.
  • Select Type (Clicks or Completes).
  • Set the limit (e.g., 50 completes).
  • Save.

Fields

Field Meaning Notes
Enabled Turns Soft Launch on/off Toggle switch
Type Whether to limit by clicks or completes Default = Completes
Limit Max number allowed before pause Required when enabled

How It Behaves

  • Project pauses automatically when limit is reached.
  • Limit applies based on selected type (clicks or completes).
  • Sends email + in-platform notification when soft launch is complete.
  • User must manually resume for full launch.
  • Can be used once per project.
