Soft Launch

Limit project exposure by sending to a small group first before going live fully.

What It Does

Lets you test a project with a limited number of respondents to catch setup issues early.

How to Enable

Note: There are multiple ways to enable Soft Launch depending on whether you're starting a new project or editing an existing one.

From Create Survey:

  • Click Start a Project in the upper-right corner of your account.
  • Choose one of the options: Get Survey Participants, Start a new survey, or Select Survey Templates.

From Edit Survey:

1

Access the Project

Navigate to the project where you want to enable the soft launch feature.

2

Select the Audience

Open the audience you want to configure for the soft launch.

3

Edit Audience Settings

  • Go to Edit Audience.
  • Navigate to the Scheduling tab.
4

Enable Soft Launch

  • Locate the "Enable Soft Launch" option and activate it.
  • Choose the soft launch type (e.g., clicks or completes).
  • Specify the number of clicks or completes required.
5

Save Changes

Click Save to apply your settings. Once saved, the soft launch configuration is active.

Fields

Field Meaning Notes
Enabled Turns Soft Launch on/off Toggle switch
Type Whether to limit by clicks or completes Default = Completes
Limit Max number allowed before pause Required when enabled

How It Behaves

  • Project pauses automatically when limit is reached.
  • Limit applies based on selected type (clicks or completes).
  • Sends email + in-platform notification when soft launch is complete.
  • User must manually resume for full launch.
  • Can be used once per project.
Important: Remember to monitor your soft launch results before proceeding with full launch. This is your opportunity to catch and fix any issues with your project setup.
